# ApCommand

JSON file to awesome print for command line tool.

dependency [awesome_print](https://github.com/michaeldv/awesome_print)

## Installation

install it yourself as:

    $ gem install ap_command

## Usage

using `ap` command.

    % ap /tmp/hoge.json
    {
      "hoge" => [
        [0] 1,
        [1] 2,
        [2] 3,
        [3] "fuga"
      ]
    }
